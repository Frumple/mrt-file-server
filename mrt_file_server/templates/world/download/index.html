Frequently Asked Questions

How do I extract these files?

  • Download and install 7-zip to extract them.
  • For multi-part files ending with the extensions .001, .002, etc., make sure you have fully downloaded all parts into the same directory, and then extract the .001 file. 7-zip will automatically combine all parts during extraction.

What is the password to extract these files?

  • The password for all world download files is: BusesTrainsShipsPlanesOhMy!

My download is slow or fails to complete!

  • Ensure that you have a stable and fast internet connection before downloading these files. A lot of these files are at least 1 GB, so expect these files to take a lot of time to fully download.

Will more world downloads be made available in the future?

  • We will try to make one world download available per year, around the time of the MRT Server's anniversary on July 13th.
