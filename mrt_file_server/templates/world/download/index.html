These world downloads are provided by two different means:
  • All New World downloads are provided via BitTorrent in order to provide faster and more reliable transfers for such large files.
  • All other world downloads can be directly downloaded from the file server by clicking the links below.

How to download the New World via BitTorrent

  1. Download and install a BitTorrent client to your local computer, such as Transmission or Deluge.
  2. In your BitTorrent client settings, ensure that encryption for incoming and outgoing connections is set to "Enabled" or "Preferred" (not "Disabled"/"Allowed" or "Forced"/"Required"). If there is also an option to set the encryption level to "Full Stream", please do so.
  3. Download the New World torrent file from one of the links below, and add the torrent to your client.
  4. You may have to wait a few minutes before the client finds other seeds and peers and starts downloading. If the download still doesn't start, you can try the following:
    1. Pause/resume the torrent several times, or have it ask/update the tracker for more peers.
    2. Check that your network/ports/router/ISP allows for torrent traffic.

How to extract the 7z file
